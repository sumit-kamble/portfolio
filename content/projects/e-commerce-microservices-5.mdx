# E-Commerce Microservices with Node.js, RabbitMQ, and MongoDB

🚀 **Built scalable and efficient microservices architecture** for an e-commerce
platform, ensuring smooth and reliable transactions.

## 🔗 [View on GitHub](https://github.com/sumit-kamble/e-commerce-microservices)

## Key Features

✅ **Microservices Architecture**  
Developed **core microservices** for:

- **User Authentication**
- **Product Management**
- **Order Processing**

✅ **Asynchronous Communication with RabbitMQ**

- Implemented **event-driven architecture** to manage **order processing and
  notifications** seamlessly.
- Ensured **message reliability and fault tolerance**.

✅ **Secure APIs with JWT Authentication**

- Used **JSON Web Tokens (JWT)** to enable **secure user authentication &
  role-based access control (RBAC)**.
- Enhanced security with **password hashing and refresh tokens**.

✅ **Core Features Implemented**

- **User Registration & Authentication**
- **Product Management (CRUD operations)**
- **Order Creation & Processing**
- **Integration with External Payment Services**

## Technologies Used

🚀 **Node.js** | 📡 **RabbitMQ** | 🗄 **MongoDB** | 🔐 **JWT Authentication** |
📦 **Docker** | 🔁 **Microservices**
